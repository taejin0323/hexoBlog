> 그냥 두서없이 블로그 만들면서 들었던 생각들을 작성

19.01.22
```
블로그에 disqus를 사용해서 댓글 기능을 추가했다.

자세히는 모르지만 최근 패치 이후 언어 선택에 한글이 빠져있는 문제점이 생겼는데
구글링 해보니 크롬 개발자 모드로 꼼수 부려서 한글이 나오게 하는 방법이 있어서 그렇게 적용해 놓았다.
```

<br>

19.01.23

```
블로그 생성한지 일주일이 지나가고 있지만 아직 모르는 것이 너무 많다ㅠ
테마를 끌어와서 내 취향대로 설정하고 싶은데 마음대로 잘 안되네...

아직까지 마크다운 사용에도 익숙하지 않고..
저번 주까지 하루패드 에디터를 사용해서 작성했지만 에디터에서 보여지는 결과물과 Github page에서 출력되는 결과물이 달라서 Typora 에디터로 바꿔봤다(하루패드는 화면이 분할되서 좋았고, Typora는 UI가 이쁘다)
일단은 plugin 기능들까지 사용하면서 최대한 나만의 개발환경을 만들어 보려고 노력중이다...

내일까지 제출해야 하는 제안서에 시간을 많이 할애하고 있는 것도 있지만,
포스팅 환경이 아직 잘 구성되지 못해서 업로드가 더딘 것 같다...

적어도 개강 전까지 최대한 빨리 적응해서 능숙해 졌으면 좋겠다
```
<br>

19.01.24
```
오늘은 알고리즘 공부했음... 언젠가 이것도 정리해서 올릴 수 있기를...

글도 얼른 정리해야하는데...
hexo에서 '== ==', '++ ++' 등 하이라이트랑 밋줄이 표시 안되는 문제가 있어서 플러그인 관련해서 조금 살펴보고 있다...
Markdown 에디터로 작성한 글이랑 서버에 올린 글이랑 다르니까 영 찝찝해서 아직 이것 저것 공부하는 중이다...
```
<br>

19.01.28
```
주말에 서울 다녀온다고 아무것도 못했다ㅠ
포스트 내 목차 링크와 각주 링크를 탔을 때, 해당 화면으로 이동하는 것 까지는 좋은데 맨 위 메뉴바가 가려서 잘 안보인다
이 문제는 어떻게 해결하는게 좋을까?...

그리고 주 마다, 하루하루 마다 목표와 계획을 짜서 해야지 계속 이랬다가는 진도가 안나가겠다
```
